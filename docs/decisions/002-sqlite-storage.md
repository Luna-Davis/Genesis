# Title: Sqlite Storage

## Context

Sqlite was used for the memory of this tool to remember the projects created.

## Decision

Sqlite was choosen due to it's ease of use and size of databases created.

## Alternatives Considered

**MySQL** was considered but due to it's heaviness in configuration it was not the first option
but might be considered in the future.

## Consequences

Sqlite was light enough to configure and start out of the box
