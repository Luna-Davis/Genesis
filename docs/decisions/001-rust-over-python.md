# Title: Rust Over Python

## Context

Rust is a systems programming language versatile for low level handling and Python is
a high level programming language.

## Decision

Rust was chosen for it's speed and low level access that it has on a system. It has a
rich ecosystem for CLI tools and with extensive features on he same.

Rust was also used as a way to learn it through project development

## Alternatives Considered

**C** was considered for this tool but lack of knowledge on the language was the main
limiter from using it.

**C++** was also considered in the creation of this project but also lack of knowledge
hindered its use.

**Python** was also considered to build it but due to its high level nature it was not
a viable candidate.

## Consequences

Going forward Rust will be the primary if not the only language in creation of **Genesis**
